# 🏒 NHL Playoff Scorer Predictor

An AI-powered model that predicts goal scorers for every game of the 2025-26 NHL Playoffs using the Anthropic Claude API.

## Features
✨ **Live 2025-26 Playoff Bracket** - Second Round through Stanley Cup Final  
🤖 **AI-Powered Predictions** - Claude analyzes matchups, team records, and top scorers  
🎯 **Per-Game Scoring** - Get predicted goals, assists, and individual player analysis  
📊 **Series Forecasts** - See who Claude thinks will win each series with confidence scores  
🎨 **Beautiful Dark UI** - Sports-themed interface with real team colors  

## Getting Started

### Prerequisites
- **Node.js 18+** - [Download](https://nodejs.org/)
- **Git** - [Download](https://git-scm.com/)
- **Anthropic API Key** - [Get one free](https://console.anthropic.com/)

### Quick Setup

```bash
# Clone the repository
git clone https://github.com/rishiquantlab/Sports-Predictions.git
cd Sports-Predictions

# Install dependencies
npm install

# Create .env file
cp .env.example .env

# Add your API key to .env
# VITE_ANTHROPIC_API_KEY=sk-ant-YOUR_KEY_HERE

# Start the dev server
npm run dev

# Open http://localhost:5173 in your browser
```

## How It Works

1. **Browse the Bracket** - View active playoff series
2. **Select a Series** - Click any matchup to get predictions
3. **AI Analysis** - Claude analyzes teams, records, and top scorers
4. **Get Results** - See predicted final score, individual scorers, and series winner

## Important: API Key Security

⚠️ **Never commit your `.env` file!** It contains your API key.

- Add `.env` to `.gitignore` (already done)
- Use environment variables on hosting platforms (Vercel, Netlify, etc.)
- Keep your API key private and rotate if accidentally exposed
